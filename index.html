This website, based on previous on work by Pamela Fox, and other researchers, lets you work on Faded Parsons Problems, a type of programming exercise where you rearrange blocks of code and fill in blanks to create a working program. You can learn more about Faded Parsons Problems in Nate Weinman's CHI 2021 paper.

The problems hosted here are all in Python 3 (via pyodide) and use drawing infrastructure provided by pyp5js, modified to use method names more similar to the py5 library for creative coding.
